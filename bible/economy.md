# 货币与资源体系版本一

## 设计原则

货币体系必须能支撑长篇而不崩塌。低阶用灵砂和下品灵石, 中阶开始灵石仍有用但不再是唯一价值, 高阶交易转向贡献、丹方、材料、洞府、灵脉权益和人情契约。

## 凡俗货币

| 货币 | 换算 | 用途 |
| --- | --- | --- |
| 铜钱 | 基础 | 白石城日常买卖 |
| 碎银 | 按重量折算, 常见一钱碎银约百枚铜钱上下 | 押金、路费、小铺周转、管事临时收费 |
| 银钱 | 1000铜钱 | 药铺、工钱、普通房租 |
| 金叶 | 100银钱 | 大额凡俗交易 |

凡俗货币可以买普通药材、粮食、布匹和人工, 但很难买真正修仙资源。

## 修仙货币

| 货币 | 换算 | 用途 |
| --- | --- | --- |
| 灵砂 | 最低单位 | 外门低阶交易, 药渣、低阶符材 |
| 下品灵石 | 100灵砂 | 引气期主流货币 |
| 中品灵石 | 100下品灵石 | 筑基、结丹前置阶段大额交易 |
| 上品灵石 | 100中品灵石 | 结丹以上稀缺流通 |
| 极品灵石 | 100上品灵石 | 不作为普通货币, 更像战略资源 |

## 白石城流通边界

白石城不是纯凡人城, 而是百草门山脚下的药材城和入山选起点。铜钱、银钱负责普通衣食、房租、脚力、凡药和日常买卖；灵砂只在丹堂、灵材边料、开脉药浴、入山选前后、低阶符材和宗门外围交易中出现。

普通街坊可以听过灵砂, 但很少真正持有灵砂。西市药铺、丹堂、商号和给修士跑腿的人会接触灵砂, 因为他们处在凡俗和修仙资源的交界处。凡俗货币不能稳定兑换灵砂, 只能通过药材、炉灰、宗门关系、任务、人情或商号折价间接换取, 以避免后期出现靠铜钱套利修仙资源的问题。

## 价格锚点

| 物品 | 价格范围 | 备注 |
| --- | --- | --- |
| 普通开脉药浴 | 120-180灵砂 | 正式弟子入门门槛, 非寻常小户能承担 |
| 稳脉药浴 | 240-360灵砂 | 富户或小修士家族常用, 降低开脉损伤 |
| 开脉散 | 80-150灵砂 | 入山选前热门, 通常不单独解决开脉 |
| 普通引气丹 | 1-3下品灵石 | 外门弟子月度核心资源 |
| 低阶净水符 | 10-20灵砂 | 任务常备 |
| 外门药童月例 | 20灵砂加半枚低阶补气丹 | 百草门入门标准 |
| 外门弟子月例 | 1下品灵石加1枚引气丹 | 需要通过考核 |
| 低阶灵兽伤药 | 40-80灵砂 | 灵兽园支线 |
| 旧丹窟入场名额 | 不公开标价 | 由贡献和执事名额控制 |

## 防通胀规则

- 不写“随手奖励十万灵石”。低阶主角见到十枚下品灵石就应有压力感。
- 灵砂不得变成凡人世界的日常通用货币。白石城凡俗账仍以铜钱和银钱为主, 灵砂只用于低阶修仙资源入口。
- 高阶交易不用单纯堆数字, 改用丹方所有权、药山采集权、宗门贡献、洞府租期、灵脉分成。
- 主角靠净尘炉能提高资源利用率, 不能凭空印钱。
- 主角不能通过小丹炉短期大量变现来凑正式弟子开脉资。低阶净粉市场容量有限, 连续出货会引来药铺、丹堂和商号追查；守砚堂父亲失踪、母亲病重、铺子衰败的家事一查便知, 突然拿出上百灵砂会直接暴露“另有隐秘货源”的风险。
- 黑市价格高但风险高, 不能成为无代价提款机。
- 宗门贡献不能完全兑换成灵石, 这样保留宗门控制力。
