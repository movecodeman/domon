# 职业体系版本二

## 炼药师

重要职业, 但不是唯一成长线。炼药师负责识药、配方、控火、成丹、验毒和丹方改良。

## 炼药师等级

炼药师品级以能稳定炼出对应品阶丹药为核心标准。所谓稳定, 不是每炉必成, 而是在材料合格、炉位合格、火阵无误的情况下, 能把对应品阶丹药炼到可入库、可售卖或可交任务的程度。

入品炼药师不等于一品炼药师。入品只能独立炼低阶药散、汤剂母粉和半成丹丸, 算是跨过学徒门槛；真正一品, 才有资格稳定开一品丹炉, 炼出可入库、可售卖或可交任务的一品丹药。

| 等级 | 能炼丹药 | 职业位置 | 正文使用边界 |
| --- | --- | --- | --- |
| 药童 | 不能独立成丹 | 洗药、分药、磨粉、守炉、记火候 | 盛砚当前 |
| 炼药学徒 | 不能独立成丹 | 可处理药材、配药粉、辅助一炉完整炼制 | 盛砚近期目标 |
| 入品炼药师 | 低阶药散、汤剂母粉和不完整丹丸 | 介于学徒和一品之间, 可参加低阶炉试 | 第一卷可出现 |
| 一品炼药师 | 一品丹药 | 小宗门低阶丹房主力 | 第一卷重点 |
| 二品炼药师 | 二品丹药 | 小宗门强势炼药师, 有独立丹房话语权 | 第一卷可出现 |
| 三品炼药师 | 三品丹药 | 小宗门核心供奉或掌门级底蕴 | 第一卷只露权威 |
| 四品炼药师 | 四品丹药 | 郡城大丹房或结丹势力核心 | 第一卷不展开 |
| 五品炼药师 | 五品丹药 | 元婴势力稀缺人物 | 第一卷不展开 |
| 六品炼药师 | 六品丹药 | 大宗长老级资源人物 | 后续大地图 |
| 七品炼药师 | 七品丹药 | 洲域级丹道强者 | 后续大地图 |
| 八品炼药师 | 八品丹药 | 顶级宗门战略人物 | 后续大地图 |
| 九品炼药师 | 九品丹药 | 传说级炼药师门槛 | 后续大地图 |

第一卷只清楚介绍药童、炼药学徒、入品炼药师、一品炼药师和二品炼药师。三品炼药师只作为掌门和宗门底蕴的权威存在, 不展开复杂规则。药王、药皇、药圣只保留为作者资料和远期悬念, 前期正文不向读者解释。

## 丹药品阶边界

| 丹药品阶 | 常见服务境界 | 说明 |
| --- | --- | --- |
| 一品丹药 | 开脉、引气前期 | 外门和低阶丹房主流, 如补气、温络、驱寒、护脉一类低阶丹药或药散 |
| 二品丹药 | 引气中后期、筑基前置 | 正式外门和内门低阶弟子常争资源 |
| 三品丹药 | 筑基期 | 百草门核心资源, 掌门和长老层面才有稳定话语权 |
| 四品丹药 | 结丹期 | 小宗门难以稳定供给 |
| 五品丹药 | 元婴期 | 青岚郡难得一见 |
| 六品以上丹药 | 化神及以上 | 后续大地图再展开 |

药童是炼药师收来协助洗药、分药、磨粉、看火、记损耗、搬药材和处理丹渣的外门外围弟子。药童未必已经开脉, 但受宗门规矩保护, 对凡人而言已算踏入仙门。优秀药童后续可获得开脉机会、月例、丹房推荐和晋升外门弟子的资格。

## 百草门炼药师定位

| 人物或职务 | 炼药师等级 | 合理性 |
| --- | --- | --- |
| 盛砚 | 药童 | 当前只会识药、分药、守炉和处理药渣, 不能独立炼药 |
| 林照晚 | 一品炼药师上等 | 能稳定炼一品丹药, 火候细, 药性判断准, 但因炉位、材料、人情和排挤迟迟无法正式冲二品 |
| 韩药师房主 | 二品炼药师 | 丹房话语权强, 能借调材料和压冷灶房, 是林照晚当前阶段的主要压力源 |
| 宋闻炉 | 一品炼药师中等 | 有独立丹房和药童, 但性格避事, 不愿与韩药师房正面冲突 |
| 叶知微 | 入品炼药师, 兼复核廊主事 | 开炉能力不是重点, 强在复核、验毒、查账和丹房秩序 |
| 百草门掌门 | 三品炼药师, 结丹境 | 百草门明面最高丹道底蕴, 能维持小宗门丹药门面, 但难以稳定触及四品丹药 |
| 陆清崖 | 曾持四品炼药师旧牌, 元婴初期 | 他不是纯炼药师, 强在丹阵、镇封和旧丹窟经验；因旧伤和封印反噬, 不能稳定开四品炉, 前期也不向盛砚显露 |

## 武修/法修

所有修士都需要基本战斗能力。百草门不是剑宗, 但外门仍教授护身体术和低阶术法, 否则采药、护送和副本任务无法生存。

第一卷重点：搬山桩、百草掌、缚藤术、引火诀、净息诀。

## 灵植师

负责灵田、药山、种植、嫁接、采收和病草处理。灵植线服务百草门日常任务，也为炼丹提供材料来源。

## 御兽师

负责灵兽饲养、兽病、兽丹、护山兽和任务灵兽。第一卷通过被丹毒污染的小灵兽引入。

## 符师

制作低阶符箓，如净水符、避瘴符、聚灵符、封火符。第一卷只作为宗门任务和副本装备出现。

## 阵师

维护药山阵法、丹房火阵和旧丹窟封印。第一卷作为旧丹窟危险的来源之一。

## 炼器师

炼制丹炉、药锄、储物袋、火钳、护身法器。第一卷只通过旧炉、炉灰和炉材引入。

## 探墟师

专门探索废弃洞府、旧宗门遗址和秘境残区。第一卷末尾埋伏笔，第二卷后可展开。

## 镇妖师

中后期出现。专门识别、封印、净化妖灵邪息。镇妖师数量稀少, 人族长期处于劣势, 因为妖灵能吞吸修士灵气壮大自身。
