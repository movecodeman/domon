# 反撞车检查

## 目的

降低与已发布中文网文明显撞车的风险。本技能不能替代正式法律审查或平台查重，但必须在发布向决策前进行创意距离检查。

## 工作流程

1. 明确要检查的元素：书名、简介、开局场景、金手指、人物、宗门、老祖暗线、剧情节点或卷纲。
2. 对照 `AGENTS.md` 和 `trackers/anti-plagiarism-log.md` 中的高风险模板。
3. 如涉及最终标题或前三章组合，使用公开来源检索。
4. 标注风险等级：低、中、高、不可接受。
5. 中风险以上必须给出改法，改核心组合，不只改名字。
6. 将检查记录写入 `trackers/anti-plagiarism-log.md`。

## 高风险信号

- 同名或近似书名。
- 同样主角起点加同样金手指功能。
- 同样前三章顺序。
- 同一特殊名词对应同一机制。
- 同样反派、家族、宗门背叛模式。
- 过度使用当前禁用模板：穿越、系统面板、夺骨、退婚、废脉星核、照灵镜测试爆炸、财务修仙。
- “普通少年加宗门选拔加古炉”属于高频区域，必须保证因果引擎、前三章结构、宗门老祖暗线和中后期妖灵功能足够不同。

## 安全差异化

- 改因果机制，不只改标签。
- 改主角付出、隐藏或失去的东西。
- 改金手指影响的社会机构。
- 改第一个不可逆选择。
- 改长期秘密的结构。
- 本项目重点保护差异点：砚山新手村地图、九曜净尘炉延迟展开为镇妖神器、稳定货币体系、妖灵吞食人族熟灵、宗门老祖隐秘扶持但不公开。

## 输出格式

返回：

- 检查元素。
- 风险等级。
- 撞车原因。
- 更安全的修订方案。
- 如使用检索，列出检索词。
