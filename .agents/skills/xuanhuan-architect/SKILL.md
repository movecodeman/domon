# 玄幻架构

## 工作流程

1. 架构修改前先读取当前核心资料：`bible/story-bible.md`、`bible/world.md`、`bible/cultivation-system.md`、`bible/combat-and-techniques.md`、`bible/economy.md`、`bible/demon-spirit-system.md`、`bible/starter-village.md`、`bible/style.md`、`bible/professions.md`、`bible/jingchen-furnace.md`、`bible/hidden-ancestor.md`、`bible/characters.md`。
2. 除非用户明确要求改路线，否则保留核心结构：普通小城少年、白石城入山选、砚山新手村、百草门小宗门、神秘远古净尘炉、宗门老祖暗中扶持、旧丹窟残痕、中后期妖灵威胁。
3. 新增设定必须能产生长期连载价值：资源、规则、势力、职业、冲突、地图层级、人物弧光或长期谜团。
4. 第一卷设定必须低视角展开，通过任务、利益、冲突和代价让读者理解，不提前解释高阶宇宙观。
5. 新增正史设定要写入对应资料文件，未兑现伏笔写入 `trackers/foreshadowing.csv` 或 `trackers/continuity.csv`。

## 架构规则

- 先让修仙世界熟悉，再让核心创意独特。
- 让读者跟着盛砚学习修仙：小城、选拔、外门、月例、灵砂、功法课、炼药杂务、砚山任务、旧丹窟外围。
- 净尘炉必须神秘且古老。前期能力是分尘、弱净火、返痕；后期才展开镇魂、封妖、炉界。
- 不让净尘炉替主角做人。主角必须观察、判断、忍耐、取舍、承担后果。
- 资源推动冲突：灵砂、灵石、宗门贡献、丹药、功法、药材、炉灰、兽材、丹方、任务名额、宗门等级。
- 主角成长不能只靠炼药，必须加入功法、体术、术法、实战、灵宠、工具、副本收获和老祖点拨。
- 妖灵体系是中后期世界压力。第一卷只露出残痕、污染、禁忌和封印。
- 宗门老祖暗线要隐秘。别人只能觉得盛砚进步快、运气好、有人照顾，不能知道老祖身份。
- 风格可以诙谐，但逻辑必须严谨；笑点来自人物和小宗门日常，不来自破坏世界规则。
- 反派要基于利益行动。坏人应保护利润、地位、秘密或制度优势。
- 每个重要女性角色先有目标、阵营、技能和压力，再谈关系张力。

## 输出格式

设计或修订架构时，输出：

- 核心决定。
- 为什么有利于长期连载。
- 新增正史设定。
- 潜在风险或防撞车注意点。
- 需要更新的资料文件或追踪表。
