# 新书工作区说明

本目录从 2026-06-11 起作为新书独立工作区。

## 清空旧书继承

旧书《归尘余烬》及此前“净尘炉路线”的人物、地名、宗门、金手指、章节、卷纲、伏笔、正文场景均视为归档资料，不再作为新书正史使用。

新书不得继承以下旧要素：

- 盛砚、白石城、砚山、百草门、守砚堂。
- 净尘炉、旧丹窟、百炉归元阵、玄殛剑、魇烬、陆清崖。
- 废丹药铺开局、入山选、药童冷灶、炉灰坡、炼药师比赛等旧剧情。

旧目录 `bible/`、`chapters/`、`outline/`、`reviews/` 和旧追踪表保留为历史归档，不参与新书设定判断。后续新书资料优先放在 `new-novel/` 下。

## 当前新书方向

新书暂定为东方玄幻修仙男频长篇。

核心方向：

- 主角从宗门练气三层外门弟子开局。
- 开局即进入外门小比失败、资源被夺、宗门制度压迫。
- 主角被拖入后山封印, 救下一只幼年守渊灵, 与其结成伴生灵契。
- 金手指核心从“残破镇渊台”升级为“伴生守渊灵”：吞煞吐灵、问灵解惑、灵影陪练、共感寻灵。
- 新书主线必须是修仙爽文：升级、斗法、任务、资源、外门逆袭和内门晋升。
