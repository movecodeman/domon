# 文风规范与发布审核表一

本文用于解决开写前缺口十和缺口十一：文风样本与审核低俗风险表。

核心目标：

- 正文要像商业修仙网文, 不是设定说明书。
- 开局要有爽文抓力, 但不低俗、不降智、不撞高危模板。
- 小岁可以有反差和口癖, 但不能刷屏。

## 一、叙事视角

采用第三人称有限视角。

主要贴沈衡视角写：

- 他看到什么。
- 他感到哪里疼。
- 他怎么算资源。
- 他如何判断风险。
- 他听见小岁在袖中或气海里传念。

不采用：

- 第一人称。
- 全知上帝大段讲史。
- 频繁“与此同时”切到反派密谋。

## 二、正文密度

每章优先写四类东西：

- 动作。
- 物件。
- 身体反馈。
- 资源变化。

少写四类东西：

- 抽象评价。
- 大段世界观。
- 空泛热血。
- 作者替人物解释性格。
- 文绉绉但不推进剧情的散文化描写。
- 需要读者停下来琢磨的奇怪比喻。

例：

不要写：

- 沈衡是一个坚韧不拔的人。

要写：

- 沈衡把气海里那点冷麻压下去, 先看齐云峥的袖口, 再看裁判执事的手。

## 三、沈衡语言规范

沈衡说话短、稳、克制。

可用句式：

- “先活下来。”
- “账不是这么算的。”
- “等交完任务再说。”
- “我输可以, 但不能连怎么输的都不知道。”
- “疼是疼, 还没到停的时候。”

禁忌：

- 不频繁喊狠话。
- 不大段自我剖白。
- 不动不动说“我要逆天”。
- 不冷酷到没有人味。

## 四、小岁语言规范

小岁特点：

- 心态幼小。
- 偶尔老成。
- 用味道理解世界。
- 说话短, 有时不完整。
- 护短、怕冷、贪吃, 但不是低龄卖萌。

可用句式：

- “漏了。”
- “这个味道坏。”
- “他抢你的。”
- “不能让它吃我。”
- “我不是灵宠。”

口癖“他娘的”规则：

- 第一卷总量三到五次。
- 不进书名、简介、章节首屏。
- 不连续出现。
- 只用于危险、怕冷、护短、委屈或反打前的小爆点。
- 这句话来自守渊灵王旧渊战场残梦, 不是现代贫嘴。

重句“守渊者, 不做祭薪”规则：

- 第一卷只在梦话或极危时露半句。
- 不解释“祭薪”。
- 中后期才成为小岁弧光核心。

## 五、斗法文风

斗法必须写清：

- 距离。
- 招式。
- 气息。
- 破绽。
- 代价。

常用写法：

- 剑尖逼近几寸。
- 气海断一息。
- 袖口灰光一闪。
- 步子慢半拍。
- 灵力只能再走一轮。

少用写法：

- 剑光纵横。
- 灵力翻涌。
- 两人大战三百回合。
- 一股恐怖气息。

斗法结果必须让读者知道：

- 为什么赢。
- 为什么输。
- 赢了花了什么。
- 输了学到什么。

## 六、开章与章末

### 开章

优先类型：

- 动作入场。
- 承接上章声音。
- 承接身体余痛。
- 承接物件变化。
- 承接未完成选择。

禁止连续同构：

- 连续用“某物落下时”。
- 连续用“不是……而是……”。
- 连续用“某人把某物如何”。
- 连续用世界观说明开头。

### 章末

章末要往前拉, 不只是停住。

可用钩子：

- 物证出现。
- 资源即将失去。
- 新任务挂出。
- 小岁传来一句关键念头。
- 对手发出下一场挑战。

禁止：

- “欲知后事如何”式生硬断章。
- 把完整世界观解释完再停。
- 情绪喊完但没有下一步行动。

## 七、人工智能感禁用清单

少用或禁用：

- “在这个弱肉强食的世界里”。
- “他不知道的是”。
- “命运的齿轮开始转动”。
- “这一刻, 他终于明白”。
- “眼神中闪过一抹坚毅”。
- “恐怖如斯”。
- “众人倒吸一口凉气”。
- “这怎么可能”刷屏。
- “废物”刷屏。
- “主角淡淡一笑”。

替代原则：

- 用具体动作写情绪。
- 用资源变化写压力。
- 用对手反应写爽点。
- 用小岁一句不完整传念写神秘。
- 用直白顺口的网文句子承载信息, 不靠晦涩修辞制造“文笔感”。

## 八、去散文化规则

本书正文是商业修仙网文, 不是散文。

写作时必须遵守：

- 文笔首先服务可读性、追读和剧情推进。
- 能直接写清楚的地方, 不绕成漂亮但费解的句子。
- 物件描写必须服务资源、危险、人物选择或伏笔。
- 比喻要少, 且必须准确、顺口、低门槛。
- 像“旧灰填进笔画里”这类只显文采、不推剧情、读起来不像正常网文作者的句子, 必须删除或改成直白表达。

精修时逐句问：

- 这一句是否让读者更想看下一句。
- 这一句是否让冲突更清楚。
- 这一句是否只是作者在装氛围。
- 这一句删掉后, 剧情和人物是否没有损失。

若删掉后没有损失, 优先删。

## 九、发布审核安全表

| 项目 | 风险 | 规则 |
|---|---|---|
| 粗口 | 中低 | “他娘的”低频, 不进简介和首屏, 不扩展其他脏话 |
| 血腥暴力 | 中低 | 前期不写虐杀和细致血腥, 妖兽战斗以动作和危险为主 |
| 人类死亡 | 中 | 入宗前和前十章不写主角杀人, 同门斗法以伤和规则为边界 |
| 邪修魔灵 | 中 | 魔影作为封印危险, 不写猎奇仪式和细致献祭 |
| 宗门腐败 | 中 | 写资源偏斜和规则缝隙, 不写现实影射口号 |
| 女性角色 | 中 | 每个重要女性有目标、资源位置和技能, 不擦边不工具化 |
| 低俗暧昧 | 低 | 第一卷不靠暧昧吸引, 重点是修炼、任务和羁绊 |
| 章节名 | 低 | 不用粗口、擦边、猎奇词做章名 |
| 简介 | 低 | 不写粗口, 不写“魔灵夺舍爽”等误导性卖点 |

## 十、正式发布前自检

每章定稿前检查：

- 首句是否接住上一章气口。
- 本章目标是否一眼清楚。
- 沈衡有没有行动选择。
- 小岁有没有帮助但不代练。
- 资源有没有获得、失去或消耗。
- 斗法或任务有没有规则支撑。
- 章末是否拉动下一章。
- 是否出现禁用模板句。
- 是否有粗口过频或低俗表达。
- 是否存在散文化、文绉绉、奇怪比喻或读者一眼不懂的句子。
- 评分是否按实际效果给出, 而不是按目标分硬抬。

第一章额外检查：

- 前三百字是否直接进入斗法。
- 是否避免世界观说明。
- 是否让沈衡差点赢。
- 是否让齐云峥作弊才赢。
- 章末灰光是否足够抓人。

## 十一、文风准备度评分

正文可执行度：9.5。

审核安全度：9.6。

小岁语言风险控制：9.5。

人工智能感控制：9.4。

说明：

文风规范已经能支撑第一章试写。但真正评分还要以样章为准, 因为设定文件无法替代正文手感。
